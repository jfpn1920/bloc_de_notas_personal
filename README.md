# 👋 ¡Bienvenidos usuarios! a mi proyecto Bloc de notas personal
<img src="imagen_presentacion.png" alt="Presentación" width="220" align="left" style="margin-right: 20px; border-radius: 5px;">
<p style="text-align: justify;">
    ¡Bienvenido a este repositorio!
    Aquí encontrarás la presentación de mi proyecto <b>Bloc de notas personal</b>, una propuesta inspirada en la organización y gestión de notas digitales mediante una interfaz moderna, intuitiva y visualmente atractiva. Este proyecto fue creado con el objetivo de representar una experiencia sencilla y organizada, enfocándose tanto en el diseño visual como en la interacción del usuario dentro de la aplicación.
</p>
<p style="text-align: justify;">
    Para el desarrollo de este proyecto se trabajó con diferentes tecnologías y enfoques de programación como <b>pseint, python, java, frontend y backend</b>, permitiendo fortalecer conocimientos en distintas áreas relacionadas con la lógica de programación, estructuras de desarrollo, diseño de interfaces y construcción de aplicaciones. Además, se aplicaron conceptos modernos como diseño responsivo, manipulación dinámica del DOM, almacenamiento local usando <b>localstorage</b> y el uso de bases de datos usando <b>phpmyadmin</b> para mejorar la experiencia visual y funcional del proyecto.
</p>
<p style="text-align: justify;">
    Este repositorio también funciona como un espacio de aprendizaje y práctica, ya que integra diferentes tecnologías y enfoques de programación relacionados con la temática <b>Bloc de notas personal</b> como ya se mensiona anteriormente. Gracias a esto, el proyecto permite fortalecer conocimientos en múltiples áreas del desarrollo mientras se mantiene una estructura organizada y orientada tanto a la lógica de programación como al diseño e implementación de aplicaciones.
</p>

### 📂 Tecnologías utilizados en este proyecto
* Pseint
* Python
* Java
* Visual basic
* Frontend
* Backend

#
### ✨ Características del proyecto
* Diseño moderno, minimalista y responsivo.
* Interfaz inspirada en un bloc de notas personal.
* Gestión dinámica y organizada de notas.
* Almacenamiento local usando <b>localstorage</b>.
* Implementación de estructuras <b>frontend</b> y <b>backend</b>.
* Uso de bases de datos mediante <b>phpmyadmin</b>.
* Aplicación de lógica de programación en diferentes tecnologías.
* Organización de ejercicios en múltiples <b>lenguajes de programación</b>.
* Proyecto desarrollado <b>sin frameworks</b>.
* Enfoque orientado al aprendizaje y fortalecimiento de habilidades tecnológicas.

#
### ⚙️ Funcionalidades implementadas
* Creación y organización de notas.
* Interfaz visual moderna y minimalista.
* Diseño adaptable a diferentes dispositivos.
* Almacenamiento local de información.
* Gestión dinámica de elementos usando JavaScript.
* Separación estructurada entre frontend y backend.
* Prácticas de lógica de programación en múltiples tecnologías.
